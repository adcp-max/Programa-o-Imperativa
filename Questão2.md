Para a Questão 2, tive dificuldades para conseguir a soma dos múltiplos de A e B, bem como a intersecção das somas. 
Para a soma do intervalo \[1, N], utilizei a soma de gauss, porém não consegui pensar em como somar os múltiplos de A e B, então, recorri a ajuda externa.
Utilizei a IA Gemini para obter clareza. Nela, aprendi que conseguia usar a fórmula de soma de uma Progressão Aritimética para resolver este problema.
Após isso, a questão prosseguiu sem problemas.
Verificando os test cases, houve um problema de overflow. Trocando as variáveis de int para long long resolveu o problema.
